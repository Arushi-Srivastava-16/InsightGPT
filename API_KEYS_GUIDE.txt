üìã STEP-BY-STEP GUIDE
================================================================================

STEP 1: Create .env File
-------------------------
Location: config/.env

Windows Command:
  copy config\env.template config\.env

Then edit config\.env with a text editor (Notepad, VS Code, etc.)


STEP 2: Neo4j Setup (REQUIRED - FREE)
--------------------------------------
What: Graph database for storing knowledge

Download: https://neo4j.com/download/

Steps:
1. Install Neo4j Desktop
2. Create New Project
3. Add Local DBMS
4. Set a password (e.g., "myPassword123")
5. Install APOC plugin (in plugins tab)
6. Start the database

Add to .env:
  NEO4J_URI=bolt://localhost:7687
  NEO4J_USERNAME=neo4j
  NEO4J_PASSWORD=myPassword123         ‚Üê Your password here


STEP 3A: Ollama Setup (RECOMMENDED - FREE)
-------------------------------------------
What: Run AI models locally on your computer
Cost: 100% FREE
Privacy: All data stays on your machine

Download: https://ollama.com

Steps:
1. Install Ollama
2. Open terminal/command prompt
3. Run these commands:
   ollama pull hermes-2-pro-llama-3-8b
   ollama pull mxbai-embed-large

Add to .env:
  LLM_PROVIDER=ollama
  OLLAMA_MODEL=hermes-2-pro-llama-3-8b
  OLLAMA_EMBEDDING_MODEL=mxbai-embed-large
  EMBEDDINGS_PROVIDER=ollama
  CHAT_PROVIDER=ollama


STEP 3B: OpenAI Setup (ALTERNATIVE - PAID)
-------------------------------------------
What: Use GPT-3.5 or GPT-4 (cloud AI)
Cost: ~$0.50 per paper (pay-as-you-go)

Get API Key: https://platform.openai.com/api-keys

Steps:
1. Create OpenAI account
2. Add payment method
3. Create API key
4. Copy the key (starts with "sk-...")

Add to .env:
  LLM_PROVIDER=openai
  OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxx  ‚Üê Your key here
  OPENAI_MODEL=gpt-3.5-turbo
  OPENAI_EMBEDDING_MODEL=text-embedding-3-small
  EMBEDDINGS_PROVIDER=openai
  CHAT_PROVIDER=openai

Pricing:
- GPT-3.5-turbo: $0.0005 per 1K tokens (~$0.50/1M tokens)
- GPT-4: $0.03 per 1K tokens
- Embeddings: $0.00013 per 1K tokens
- Typical usage: $5-20/month for regular use


STEP 4: Zotero Setup (OPTIONAL - FREE)
---------------------------------------
What: Import PDFs from your Zotero library
Cost: FREE

Get API Key: https://www.zotero.org/settings/keys

Steps:
1. Log in to Zotero
2. Go to Settings ‚Üí Feeds/API
3. Find your User ID
4. Create a new API key (read permissions)
5. Find your Zotero storage folder

Add to .env:
  ZOTERO_ENABLED=true
  ZOTERO_LIBRARY_ID=12345678              ‚Üê Your User ID
  ZOTERO_LIBRARY_TYPE=user
  ZOTERO_API_KEY=xxxxxxxxxxxxx            ‚Üê Your API key
  ZOTERO_STORAGE_DIR=C:\Users\YourName\Zotero\storage\

================================================================================
üÜò TROUBLESHOOTING
================================================================================

Problem: "Cannot find .env file"
Solution: Make sure file is named .env (not .env.txt)
          Location should be: config/.env

Problem: "Neo4j connection failed"
Solution: - Check Neo4j Desktop is running
          - Verify password in .env matches Neo4j
          - Check URI: bolt://localhost:7687

Problem: "OpenAI API error"
Solution: - Verify API key is correct
          - Check you have credits
          - Try switching to Ollama (free)

Problem: "Ollama not found"
Solution: - Install from https://ollama.com
          - Run: ollama serve
          - Pull models again

Problem: "Import errors"
Solution: pip install -r requirements.txt --upgrade
